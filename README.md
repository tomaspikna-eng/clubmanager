# CSP Club Manager — nový samostatný frontend

Samostatný statický frontend pre vlastný GitHub repozitár a vlastný Vercel projekt.

## Nasadenie
1. Nahraj celý obsah ZIP do koreňa nového GitHub repozitára.
2. Pripoj repozitár vo Verceli.
3. Framework preset: Other.
4. Build command: prázdny.
5. Output directory: prázdny.
6. Pripoj samostatnú doménu, napr. cspmanager.app.

## Prístup
Povolené plány: `ultra`, `elite`, alebo `is_admin = true`.

## Funkčné moduly vo verzii 0.1
- login cez spoločný CSP Supabase Auth,
- databázový bootstrap a kontrola plánu,
- live dashboard,
- štart/pauza/pokračovanie/stop,
- výpočet ceny,
- potvrdenie do kasy,
- zoznam a vytvorenie staníc.


## GitHub Pages
Súbory musia byť priamo v koreňovom adresári repozitára. ZIP je pripravený bez nadradeného priečinka a podporuje aj project path, napr. `/clubmanager/`.
